# Instance Segmentation - UNet

## Dependencies
### (Tested Tensorflow-GPU ENV with 3080)
- Python = 3.7.16
- Tensorflow-GPU = 2.5.0
- CudaToolKit = 11.3.1
- CuDNN = 8.2.1
- NumPY
- Pandas
- MatPlotLib
- Segmentation-models
- Scikit-Learn

This is a model for instance segmentation, based off UNet with a ResNet backbone.
